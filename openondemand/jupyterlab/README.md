# JupyterLab Examples

Examples for the JupyterLab tutorial on KENET HPC.

## Files

- `00_package_installation.sh` - Installing Python packages
- `01_basic_plotting.py` - Basic plotting example
- `02_loading_data.py` - Loading data from cluster storage
- `03_gpu_tensorflow.py` - Using GPU with TensorFlow
- `04_dask_parallel.py` - Parallel processing with Dask
- `05_plotly_interactive.py` - Interactive visualizations with Plotly

## Workflows

Complete workflow examples:
- `workflows/01_data_science_pipeline.py` - Complete data science workflow
- `workflows/02_ml_experiment.py` - Machine learning experiment
- `workflows/03_geospatial_analysis.py` - Geospatial data analysis

## Usage

Copy these examples to your JupyterLab session and modify as needed for your work.
