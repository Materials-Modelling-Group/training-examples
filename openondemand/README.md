# KENET Open OnDemand Tutorials - Code Examples

This repository contains all code examples referenced in the KENET HPC Open OnDemand tutorials.

## Repository Structure

- `jupyterlab/` - JupyterLab examples
- `vscode/` - VS Code examples
- `mlflow/` - MLflow experiment tracking examples
- `tensorboard/` - TensorBoard visualization examples
- `octave/` - GNU Octave examples
- `qgis/` - QGIS geospatial examples
- `pspp/` - PSPP statistical analysis examples

## Usage

Each directory contains example code files referenced in the corresponding tutorial on the KENET wiki.

Wiki: https://training.kenet.or.ke/index.php/Jupyter
## Getting Started

Clone this repository to your KENET HPC home directory:

```bash
cd /home/username
git clone https://github.com/Materials-Modelling-Group/training-examples.git
cd ood-tutorials
```
