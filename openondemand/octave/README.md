# GNU Octave Examples

Examples for the Octave tutorial on KENET HPC Remote Desktop.

## About Octave

GNU Octave is a high-level programming language primarily intended for numerical computations. It is largely compatible with MATLAB.

## Directory Structure

- `examples/` - Basic Octave examples covering fundamental operations
- `workflows/` - Complete workflows for engineering and scientific applications

## Requirements

- GNU Octave 8.x or later
- Octave packages: signal, image, statistics (install as needed)

## Getting Started

1. Launch Remote Desktop from Open OnDemand
2. Open Octave GUI or command line
3. Navigate to this directory
4. Run examples in order

## Installing Packages
```octave
pkg install -forge signal
pkg install -forge image
pkg load signal
```
