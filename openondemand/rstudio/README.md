# RStudio Server Examples

Examples for the RStudio Server tutorial on KENET HPC.

## Directory Structure

- `examples/` - Basic examples covering fundamental RStudio tasks
- `workflows/` - Complete analysis workflows and project templates

## Requirements

- R version 4.5.2 or later
- RStudio Server on KENET HPC cluster
- Recommended packages: tidyverse, rmarkdown, knitr

## Usage

Open RStudio Server through Open OnDemand and load these scripts to explore different features and workflows. Modify the examples for your own research needs.

## Getting Started

1. Launch RStudio Server from Open OnDemand
2. Set working directory: `setwd("/home/username/training-examples/rstudio")`
3. Open any example file to begin
